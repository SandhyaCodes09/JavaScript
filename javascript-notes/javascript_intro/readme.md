# Introduction to JavaScript

JavaScript is a high-level, interpreted programming language used to make web pages interactive.
It was initially created to run inside web browsers, but today it can also run on servers using Node.js.

## What JavaScript can do
- Create dynamic and interactive websites
- Validate forms on the client side
- Update content without reloading the page
- Build web and server-side applications

## Where JavaScript is used
- Web Browsers (Chrome, Firefox, Edge)
- Server-side development (Node.js)
- Mobile applications
- Web-based games

## Features of JavaScript
- Easy to learn
- Lightweight
- Interpreted language
- Object-based programming language
- Platform independent

## JavaScript is used with
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic and Interactivity)
